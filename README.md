# WordGame (Java)

A console-based word guessing game where the user selects a category and attempts to guess the word within a limited number of tries.

## Features
- Multiple categories (Animals, Food, Movies)
- Tracks guessed letters
- Displays remaining guesses
- Win and game over conditions

## How It Works
1. User selects a category
2. A random word is chosen
3. User guesses letters
4. Game continues until the word is guessed or attempts run out
